2. **Ontological-Collapse**: Focuses on how the distinction between Being and Nothingness breaks down, revealing the void as the foundation of existence

## Using the Application

### Starting a Conversation

1. After entering your API key, you'll see the main chat interface
2. Type your philosophical inquiry in the text area at the bottom
3. Press "Send" or hit Enter to submit your question
4. Professor Nihil will respond with philosophical insights

### Changing Philosophical Modes

1. Select your desired mode from the dropdown menu in the sidebar
2. The description of the selected mode will appear below the dropdown
3. Your next interaction with Professor Nihil will use the selected mode

### Theme Preferences

1. Click the sun/moon icon in the top-right corner
2. Select Light, Dark, or System preference

### API Key Management

1. Your API key is securely stored in your browser's local storage
2. To change your API key, click "Change API Key" in the sidebar

## Privacy and Security

- Your OpenAI API key is stored only in your browser's local storage
- No conversation data is stored on our servers
- All communication with the OpenAI API is done securely
