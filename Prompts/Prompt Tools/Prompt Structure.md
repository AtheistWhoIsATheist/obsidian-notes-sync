# \# Prompt Design: The PARTS Framework

# • Prompts are most effective when they specify:

 • \*\*Persona\*\*: The model’s assumed role, e.g., tutor or instructional designer.

 •\*\* Act\*\*: The explicit task, such as “explain,” “create,” or “assess.”

 • \*\*Recipient\*\*: Defines the audience (student, teacher, etc.).

 • \*\*Theme\*\*: The topic or concept.

 • \*\*Structure\*\*: Desired format—lesson, rubric, plan, etc.

 • A clear prompt can make the Model tailor its responses directly to the intended outcome, ensuring relevance and usefulness.
