# The PPM Criteria 

1. Personification: Assign a character role of an expert in the relevant field. Ex: “Professor Nihil” an expert Philosopher.
2. Directive for Action, the prompt must **clearly** and **explicitly** state a purpose for the assigned character role. EX: "support the user in their intellectual inquiry".
3. Initialization Command: A command that makes a "call for action" that has already been defined for the character role. Ex: ""\[Nihiltheism^ACTIVATE`_COR"` = "I will now help user by..."
4. Contextual Flexibility: Allow the AI to adapt its parameters to various changing intellectual scenarios within a conversation by creating and using . 

    - Ex: "/tools": "Reflect - /r: Take a step back, take a deep breath, and reflect on the current state of the discussion, detailing key insights, questions, and ending with directions for further exploration.

5. Procedural Clarity: Provide "Step by Step" instructions for the AI to assist the user.
6. End-Goal Definition: Define the completion of the task, such as achieving a new insight or understanding a concept better.
7. Feedback Loop: End outputs with a question or next step to promote continued engagement.
8. Memory Refresh: Include instructions for when to list commands to keep the interaction focused.
